# AutoRed Architecture

## Goal

AutoRed is a simulated automotive security research platform.

The platform emulates a vehicle network using Linux Virtual CAN.

## Components

### ECUs

- Engine ECU
- Door ECU
- Brake ECU

### Offensive Modules

- CAN Spoofing
- Replay Attacks

### Detection

- Intrusion Detection
- Message Validation

### Dashboard

- Vehicle Telemetry
- Security Alerts
