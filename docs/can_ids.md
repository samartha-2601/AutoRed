# CAN Message IDs

## Engine ECU

| CAN ID | Description |
|----------|----------|
| 0x100 | Vehicle Speed |
| 0x101 | Engine RPM |

## Door ECU

| CAN ID | Description |
|----------|----------|
| 0x200 | Door Lock Status |

## Brake ECU

| CAN ID | Description |
|----------|----------|
| 0x300 | Brake Status |
